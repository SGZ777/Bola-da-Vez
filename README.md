# Bola-da-Vez 
O Bola da Vez é um projeto simples de site voltado para o tema esportivo, desenvolvido como meu primeiro contato com desenvolvimento web. Ele foi criado durante a fase inicial de aprendizado, com foco nos fundamentos de HTML e CSS, sem a utilização de frameworks ou bibliotecas externas.

O objetivo do projeto foi compreender conceitos básicos como estruturação de páginas, uso de tags semânticas, organização de conteúdo e estilização com CSS puro. Por ser um projeto inicial, sua proposta é simples e voltada exclusivamente para aprendizado.

O Bola da Vez marcou o início da minha jornada no desenvolvimento web, servindo como base para projetos posteriores e para a evolução do meu entendimento sobre a criação de interfaces.
